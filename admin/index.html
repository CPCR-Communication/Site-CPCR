Paramètres

Mot de passe

Le mot de passe de connexion se modifie dans le tableau de bord Netlify : Site configuration → Environment variables → ADMIN_PASSWORD. Demandez à la personne qui gère le site technique si vous souhaitez le changer.

Aide rapide

  • Modifiez ce que vous souhaitez, puis cliquez sur Enregistrer en haut à droite.
  • Pour les catalogues : changez le nom, l'image et le lien, puis réorganisez par glisser-déposer.
  • Les images envoyées sont enregistrées dans images/uploads/.
  • Les textes sont mis à jour immédiatement. Les nouvelles images peuvent prendre 1 à 2 minutes.
